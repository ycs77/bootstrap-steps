Bootstrap Steps

A simple Bootstrap 4 steps plugin, support mobile client.

Github

Install:

$ npm install bootstrap-steps
or
$ yarn add bootstrap-steps

Example:

  • 1 Step 1
  • 2 Step 2
  • 3 Step 3
  • 4 Step 4
